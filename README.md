# Mini POS System

A Point of Sale (POS) system built with Django while learning web development.

## Features

- Student authentication
- Inventory management
- Sales management
- Django Admin
- MySQL Database

## Technologies Used

- Python
- Django
- MySQL
- HTML
- CSS

## Installation

```bash
git clone <repository-url>
cd POS

python -m venv env
source env/bin/activate   # macOS/Linux

pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
